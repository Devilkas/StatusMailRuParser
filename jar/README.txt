<--------------------------------------StatusMailRuParser-------------------------------------->

Программа предназначина для проверки статусов ONLINE/OFFLINE электронной почты с сайта mail.ru

ИНСТРУКЦИЯ:
1. Внести список email (электронной почты с сайта mail.ru) в файл "bin/emails.ini" (КАЖДЫЙ email С НОВОЙ СТРОКИ)
2. Запустить файл start.bat
3. В файле logs.log находиться статусы отслеживаемых email
4. Программа работает до тех пок пока не будет закрыта окно командной строки cmd запущеное файлом start.bat
